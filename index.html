The MadLibs game was built using HTML, CSS, and JavaScript. I created the story paragraph by using temple literals which made it easier to combine the string. I used CSS Flexbox for the design and formatting. I had a problem with the spacing. There was some kind of conflict and it would not respond to a change in the CSS so I moved the change to the html file, inline and that fixed it.The interactivity and instant feedback made it a fun project to build.The result is displayed dynamically, allowing users to instantly see their custom story.

About Me

Click Here for About Me

For my About Me website, I used a combination of HTML and CSS create a simple yet engaging page that provides insight into my background. I incorporated CSS animations to add a dynamic touch and used flexbox for a well-structured layout. I also used well placed colors to give the illusion of a card turning without using javascript. This approach made the site both visually appealing and easy to maintain One challenge was finding the right changes to make for responsiveness. I used the CSS settings under margin to find the right balance between design and readability. I enjoyed finding how to add animations to make the site more engaging with only CSS
